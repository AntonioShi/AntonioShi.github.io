特殊矩阵的相关性质

字数统计: 507字   |   阅读时长: 2分


矩阵的迹tr:

在线性代数中，n乘n方阵“A”的迹，是指“A”的主对角线各元素的总和（从左上方至右下方的对角线），例如：

性质:

设有N阶矩阵A，那么矩阵A的迹（用tr(A)表示）就等于A的特征值的总和，也即矩阵A的主对角线元素的总和。

实对称矩阵

矩阵的秩的性质：

设A是sn矩阵，B为ns矩阵，且AB=0，则r(A) + r(B) <= n

证: 将B按列分块为 B=(b1,…,bs)
因为 AB=0
所以 A(b1,…,bs) = (Ab1,…,Abs)=0
所以 Abi=0, i=1,…,s
即 B 的列向量都是齐次线性方程组 AX=0 的解向量
所以B的列向量组可由 AX=0 的基础解系线性表示
而 AX=0 的基础解系含 n-r(A) = n-r 个向量
所以 r(B) <= n-r.

所以r(A)+r(B)<=n
